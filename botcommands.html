ChatSage Bot Commands

Click any command row below to expand and see usage details.
Command Description
General
!ask / !sage Ask Sage a question. Uses search intelligently.▼
!ask <your question>
!search Searches the web for information on a topic.▼
!search <your query>
!game Provides researched information about the game currently being played. Add "analyze" to use AI image recognition on the stream (uses Twitch thumbnail that updates every 5 minutes).▼
!game !game analyze (analyzes stream visuals with AI) !game <your question> (answers specific gameplay questions) !game analyze <your question> (analyzes visuals + answers question)
!translate Manage automatic message translation for users.▼
!translate <lang> !translate stop !translate <lang|stop> [user] !translate stop all
!commands / !help Shows where to find the list of available commands.▼
!help or !commands
!ping Checks if the bot is responsive. Responds with Pong!▼
!ping
Config
!botlang Configure the bot to speak in a specific language for the entire channel.▼
!botlang <language> !botlang status (check current language) !botlang off (reset to English) !botlang default (reset to English)
Games
!geo Starts or manages Geo-Game, a geography guessing game for IRL or virtual worlds.▼
!geo (start global real-world game) !geo <region> (start region-specific real-world game, e.g. !geo Japan) !geo game (start game-world game using current stream's game) !geo game [Game Title] (start game-world game with specified game, e.g. !geo game "Metroid Prime") !geo <command> N (run any game command with N rounds, e.g. !geo Japan 3) !geo leaderboard !geo stop (game initiator, mods, or broadcaster can stop) !geo config difficulty <easy|normal|hard> !geo config interval <seconds> (30-300s between clues) !geo config duration <minutes> (3-20m per round) !geo config region <regions> (e.g. Europe,Asia or USA,Canada) !geo config game <titles> (preferred game titles, comma-separated) !geo config pointsbase <number> (1-100) !geo config pointstimebonus <true|false> !geo config pointsdifficultymultiplier <true|false> !geo config scoretracking <true|false> !geo config maxrounds <number> (e.g., 1-10) !geo resetconfig !geo clearleaderboard (or resetstats) !geo report <reason> (report problematic locations or clues) !geo help
!riddle Starts or manages a Riddle game. Riddles can be general, about the current stream game, or a specific subject.▼
!riddle (starts a general riddle) !riddle <subject> (e.g., !riddle animals) !riddle <rounds> (e.g., !riddle 3 for 3 rounds) !riddle <subject> <rounds> (e.g., !riddle history 5) !riddle game (riddle about the current stream game) !riddle game <rounds> !riddle leaderboard !riddle stop (game initiator, mods, or broadcaster can stop) !riddle config difficulty <easy|normal|hard> !riddle config questiontime <seconds> (15-120s per riddle) !riddle config pointsbase <number> (1-100) !riddle config pointstimebonus <true|false> !riddle config pointsdifficultymultiplier <true|false> !riddle config scoretracking <true|false> !riddle config maxrounds <number> (e.g., 1-20) !riddle config keywordslimit <number> (recent keywords to exclude, e.g. 20-100) !riddle config rounddelay <milliseconds> (delay between rounds) !riddle resetconfig !riddle clearleaderboard (or cleardata) !riddle help
!trivia Starts or manages Trivia games with factual questions on various topics, including the current stream game.▼
!trivia (start general knowledge trivia) !trivia <topic> (start topic-specific trivia, e.g. !trivia science) !trivia game (start trivia about the current stream game) !trivia <rounds> (specify number of rounds, e.g. !trivia 3) !trivia <topic> <rounds> (e.g. !trivia history 5) !trivia leaderboard !trivia report <reason> (report problematic questions) !trivia stop (game initiator, mods, or broadcaster can stop) !trivia config difficulty <easy|normal|hard> !trivia config time <seconds> (10-120s per question) !trivia config pointsbase <number> (1-100) !trivia config pointstimebonus <true|false> !trivia config pointsdifficultymultiplier <true|false> !trivia config scoretracking <true|false> !trivia config topic <topics> (preferred topics, comma-separated) !trivia config maxrounds <number> (e.g., 1-10) !trivia resetconfig !trivia clearleaderboard !trivia help
