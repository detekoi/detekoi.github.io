StreamSage Bot Commands

Command Description Usage
!ask / !sage Ask Sage a question. Uses search intelligently. !ask <your question>
!commands / !help Shows where to find the list of available commands. !help or !commands
!game Provides researched information about the game currently being played. Add "analyze" to use AI image recognition on the stream. !game !game analyze (analyzes stream visuals with AI) !game <your question> (answers specific gameplay questions) !game analyze <your question> (analyzes visuals + answers question)
!search Searches the web for information on a topic. !search <your query>
!translate Manage automatic message translation for users. !translate <lang> !translate stop !translate <lang|stop> [user] !translate stop all
!geo Starts or manages Geo-Game, a geography guessing game for IRL or virtual worlds. !geo (start global real-world game) !geo <region> (start region-specific real-world game, e.g. !geo Japan) !geo game (start game-world game using current stream's game) !geo game [Game Title] (start game-world game with specified game, e.g. !geo game "Metroid Prime") !geo <command> N (run any game command with N rounds, e.g. !geo Japan 3) !geo leaderboard !geo stop (game initiator, mods, or broadcaster can stop) !geo config difficulty <easy|normal|hard> !geo config interval <seconds between clues> !geo config duration <length of game in minutes> !geo config region <comma-separated regions> (e.g. Europe,Asia or USA,Canada,Mexico) !geo resetconfig !geo clearleaderboard !geo resetstats (also clears leaderboard) !geo report <reason> (report problematic locations or clues) !geo help
!trivia Starts or manages Trivia games with factual questions on various topics, including the current stream game. !trivia (start general knowledge trivia) !trivia <topic> (start topic-specific trivia, e.g. !trivia science) !trivia game (start trivia about the current stream game) !trivia <rounds> (specify number of rounds, e.g. !trivia 3) !trivia <topic> <rounds> (e.g. !trivia history 5) !trivia leaderboard !trivia report <reason> (report problematic questions) !trivia stop (game initiator, mods, or broadcaster can stop) !trivia config difficulty <easy|normal|hard> !trivia config time <seconds> (time per question) !trivia config points <value> (base points per correct answer) !trivia resetconfig !trivia clearleaderboard !trivia help
!ping Checks if the bot is responsive. Responds with Pong! !ping
