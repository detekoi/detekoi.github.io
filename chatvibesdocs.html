ChatVibes TTS Bot Documentation

Commands

Click any command row below to expand and see usage details.
Command Description
General
!tts status Gets the current status of the TTS application for the channel. ▼
!tts status

Permission: Everyone
!tts voices Shows a summary of available voices. ▼
!tts voices

Permission: Everyone
!tts commands Lists all available !tts subcommands. ▼
!tts commands

Permission: Everyone
Engine & Mode Control (Moderator Only)
!tts on / enable Enables the TTS engine. ▼
!tts on !tts enable

Permission: Moderator
!tts off / disable Disables the TTS engine entirely. ▼
!tts off !tts disable

Permission: Moderator
!tts mode Toggles TTS mode (all chat or command only). ▼
!tts mode all (Speak all chat messages & events) !tts mode command (Only speak on command or for events) !tts mode (Displays current mode)

Permission: Moderator
Queue Management (Moderator Only)
!tts pause Pauses the TTS event queue. ▼
!tts pause

Permission: Moderator
!tts resume Resumes the TTS event queue. ▼
!tts resume

Permission: Moderator
!tts clear Clears all pending messages from the TTS queue. ▼
!tts clear

Permission: Moderator
!tts stop Stops the currently playing audio. ▼
!tts stop

Permission: Moderator
User & Event Preferences
!tts emotion Set your preferred TTS emotion. ▼
!tts emotion <emotion_name> (e.g., happy, sad, angry, auto) !tts emotion reset (Use channel default) !tts emotion (Displays current personal preference)

Valid emotions: auto, neutral, happy, sad, angry, fearful, disgusted, surprised.

Permission: Everyone (for their own preference)
!tts voice <voice_id|reset> Allows a user to set their preferred voice for messages they trigger that are spoken by TTS. Use reset to revert to the channel's default voice. See available voices for a list of voice IDs. ▼
!tts voice Friendly_Person !tts voice reset !tts voice (displays current personal preference)

Permission: Everyone (for their own preference)
!tts ignore Manage TTS ignore list. ▼
!tts ignore add <username> !tts ignore del <username>

Permission: Moderator
!tts ignored Lists users on the TTS ignore list. ▼
!tts ignored

Permission: Moderator
!tts events Toggles TTS for Twitch events (subs, cheers, etc.). ▼
!tts events on !tts events off !tts events (Displays current status)

Permission: Moderator
Voice Configuration (Moderator Only)
!tts defaultvoice <voice_id|reset> Sets the default TTS voice for the channel. Use reset to revert to the system default. Use !tts voices to see available types/get a link to IDs. ▼
!tts defaultvoice Friendly_Person !tts defaultvoice reset

Permission: Moderator

Usage: !tts defaultvoice Friendly_Person (Use a valid Voice ID from the minimax/speech-02-turbo model)

Note: This sets the channel-wide default voice. Individual users can still set their own preferred voice with !tts voice <voice_id>.
!tts speed <0.5-2.0> (Conceptual) Sets the default speech speed. ▼
!tts speed 1.2 (1.0 is normal)

Permission: Moderator
!tts pitch <-12-12> (Conceptual) Sets the default speech pitch. ▼
!tts pitch 2 (0 is normal)

Permission: Moderator
Direct TTS (Moderator Only)
!tts say <message> Enqueues the provided message for TTS. ▼
!tts say Welcome everyone to the stream!

Permission: Moderator
